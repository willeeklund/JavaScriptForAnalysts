Equality Operators (==)

 

print("hej" == "hej2");

 


print('' == '0')           // false
print(0 == '')             // true
print(0 == '0')            // true

print(false == 'false')    // false
print(false == '0')        // true

print(false == undefined)  // false
print(false == null)       // false
print(null == undefined)   // true

print(' \t\r\n ' == 0)     // true

Always use === and !==

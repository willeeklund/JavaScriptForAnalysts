Equality Operators (==)

log('' == '0')           // false
log(0 == '')             // true
log(0 == '0')            // true

log(false == 'false')    // false
log(false == '0')        // true

log(false == undefined)  // false
log(false == null)       // false
log(null == undefined)   // true

log(' \t\r\n ' == 0)     // true

Always use === and !==

All of the comparisons just shown
produce false with the === operator.
